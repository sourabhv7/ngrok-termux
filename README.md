# termux-ngrok
O [ngrok](https://ngrok.com/) 
(https://ngrok.com/docs).

### Install ngrok on Termux
```
apt update && apt upgrade
apt install git
git clone https://github.com/sourabhv7/ngrok-termux.git
cd termux-ngrok
chmod +x termux-ngrok.sh
./termux-ngrok.sh
```
